# MICB405_Bioinformatics_Team7

* All **supplementary materials for the MICB 405 Final Report**, including R scripts, raw data tables, and other relevant files, are available in the [Supplementary_Final_Report](./Supplementary_Final_Report/) directory.
* This project assesses the transcriptomic impact of LPS stimulation in primary human monocytes as a model for rheumatoid arthritis progression.
